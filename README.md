# Sentiment-Analysis
This project performs sentiment analysis on textual data and generates a word cloud visualization of negative sentiments to uncover key patterns and insights.


Overview :

The workflow involves analyzing text data using Natural Language Processing (NLP) techniques and visually highlighting the most frequent words in negative reviews. This helps in identifying common issues, pain points, or themes present in the dataset.

🔧 Key Features
Performs sentiment classification using polarity scoring
Categorizes text into Positive, Negative, and Neutral
Filters and analyzes negative sentiments
Generates an intuitive word cloud visualization
Simple and beginner-friendly implementation using Python


🛠️ Tech Stack :
Python
Pandas
TextBlob (for sentiment analysis)
WordCloud
Matplotlib


📌 Workflow :
Load dataset containing textual data
Apply sentiment analysis using TextBlob
Classify each sentence into sentiment categories
Extract negative reviews
Generate and visualize a word cloud of negative terms


📈 Use Cases :
Customer review analysis
Product feedback insights
Social media sentiment tracking
Identifying key issues from user feedback
💡 Insights

The word cloud helps quickly identify the most common negative keywords, enabling better decision-making and targeted improvements.
